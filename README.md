# @siegblink/starwars-name-generator
Get a random Star Wars name.

### To generate a random name.
```
const starWarsNames = require('@siegblink/starwars-name-generator')
starWarsNames.random()
```

### To view all available names.
```
const starWarsNames = require('@siegblink/starwars-name-generator')
starWarsNames.all
```
